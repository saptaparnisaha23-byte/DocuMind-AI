## 📌 Project Summary

DocuMind-AI is an enterprise-grade Retrieval-Augmented Generation (RAG) platform that turns PDF documents (textbooks, research papers, course syllabi) into an interactive, grounded knowledge base.

### 🌟 Key Technical Highlights & Metrics
1. **Hybrid Vector & Sparse Search:** Combines ChromaDB HNSW dense vector search with BM25 keyword matching using Reciprocal Rank Fusion ($k=60$).
2. **Adaptive Response Detail Routing:** Generates clean 2–3 line definitions for "What is" queries and structured multi-section breakdowns for "Explain in detail" requests.
3. **Multi-Document Comparative Analysis:** Compares multiple PDFs side-by-side with exact page numbers and inline quote citations.
4. **Anti-Hallucination Guardrails:** Enforces strict evidence thresholds to trigger clean refusal on out-of-corpus questions.
5. **20-Q&A Benchmark Scores:** Factual Correctness (4.90/5) | Citation Precision (4.95/5) | Completeness (4.85/5).
